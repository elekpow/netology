Шифрование RSA
ключ:  {5, 21}
------------------------------
n = p * q = 105.
F= (p-1) * (q-1)  = 80
e= 11
открытый ключ{e, n} = {11, 105}
вычислим d из формулы “(d * е) mod F =1” / d = 51
закрытый ключ {d, n}={51, 105}
----------------------------------
m - текст для шифрования  (число 7)
----------------------------------
(m^e) mod n = c
шифруем  (7 ^ 11) mod 105 = 52
(c^d) mod n = m
расшифруем  (52 ^ 51) mod 105 = 7
